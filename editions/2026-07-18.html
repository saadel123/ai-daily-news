💡 Hack des Tages · Tip of the day

Claude Code 2.1.214 · EndConversation · Docker-Sicherheit
Deutsch

Claude Code 2.1.214 bringt zwei Sicherheitsänderungen, die im Alltag direkt spürbar werden: Erstens kann Claude jetzt Sessions selbst beenden, wenn ein User massiv missbraucht oder aktiv jailbreakt — das EndConversation-Tool war auf claude.ai seit 2025 aktiv und landet jetzt auch in der CLI. Zweitens werden Docker-Befehle mit Daemon-Redirect-Flags wie --url, --connection oder --identity ab sofort durch einen Permission-Prompt abgefangen — bisher liefen diese unbemerkt durch. Dazu kommt ein Rolling-Progress-Heartbeat, der bei lang laufenden Tool-Calls jetzt sichtbar tickt.

English

Claude Code 2.1.214 ships two security changes you'll notice right away: first, Claude can now end sessions on its own when a user is seriously abusive or actively jailbreaking — the EndConversation tool has been live on claude.ai since 2025 and now lands in the CLI too. Second, docker commands carrying daemon-redirect flags like --url, --connection, or --identity are now intercepted with a permission prompt — previously these ran silently without warning. Bonus: a rolling progress heartbeat that visibly ticks during long-running tool calls.

BEFEHL · COMMAND
docker [--url | --connection | --identity] [command]
podman --remote [command]
--url
connect to a Docker daemon at a custom TCP or socket URL — now requires approval in Claude Code
--connection
use a named Docker context or connection — now requires approval in Claude Code
--identity
authenticate with a named identity — now requires approval in Claude Code
--remote
Podman remote mode against a remote API — now requires approval in Claude Code

# Podman remote ebenfalls jetzt gesichert · Podman remote is now gated too
