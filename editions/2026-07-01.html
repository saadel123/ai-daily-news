💡 Hack des Tages · Tip of the day

Claude Code 2.1.196 · Org-Standardmodell in /model
Deutsch

Claude Code 2.1.196 bringt Organisations-Standardmodelle direkt ins Terminal: Admins legen das Modell einmal im Org Console fest — danach zeigt der Slash-Command für jeden im Team automatisch "Org default" oder "Role default" an, wenn kein persönliches Modell gewählt ist. Dazu gibt es jetzt lesbare Standardnamen für Sessions, die das Wiederfinden und Anschreiben von Hintergrund-Agenten deutlich einfacher macht. Und ein kleines UX-Upgrade: Cmd- bzw. Strg-Klick auf Datei-Anhänge im Chat öffnet sie direkt im Finder oder Explorer.

English

Claude Code 2.1.196 brings org-level default models right into your terminal: admins set the model once in the org console, and from then on the slash command shows "Org default" (or "Role default") for everyone who hasn't picked their own. On top of that, sessions now get readable default names, making background agents much easier to find and message. And a small UX win: Cmd/Ctrl-click on file attachments in chat opens them directly in Finder or Explorer.

BEFEHL · COMMAND
/model
# Org-Standardmodell sehen, wenn kein persönliches gewählt ist · See "Org default" when no personal model is selected
$ /model

# Persönliches Modell für die Session wählen · Pick a model for this session only
$ /model sonnet-5
