Claude Code 2.1.149 · /model Session · /resume bg · GFM-Checkboxen
Deutsch

Claude Code bringt in den Versionen 2.1.144 und 2.1.149 ein feines Paket an Verbesserungen für den Entwickleralltag. Der /model-Picker ist jetzt sitzungsgebunden — willst du den globalen Default ändern, drückst du d im Picker, ohne die laufende Session zu berühren. Der /resume-Befehl nimmt jetzt auch Hintergrundagenten mit auf: Sessions aus claude --bg oder der Agenten-Ansicht tauchen in der Auswahl-Liste auf, klar als bg gekennzeichnet. Außerdem rendert Markdown-Output jetzt GFM-Checkboxen als echte, anklickbare Kästchen — ideal für Task-Listen in langen Sessions. Enterprise-Teams können mit dem neuen allowAllClaudeAiMcps-Setting alle claude.ai Cloud-MCP-Konnektoren neben der managed-mcp.json einbinden.

English

Claude Code versions 2.1.144 and 2.1.149 ship a tidy bundle of developer quality-of-life improvements. The /model picker is now session-scoped — pressing d inside the picker sets the global default without touching the current session. /resume now picks up background sessions too: sessions started via claude --bg or agent view appear in the picker, clearly marked as bg. Markdown output now renders GFM task list checkboxes as real, clickable boxes — great for task lists in long-running sessions. Enterprise teams get a new allowAllClaudeAiMcps managed setting to load all claude.ai cloud MCP connectors alongside managed-mcp.json.
