Google Gemini · Managed Agents mit Background-Tasks und Remote-MCP
Deutsch

Google hat die Managed Agents in der Gemini API erheblich ausgebaut: Agent-Interaktionen laufen jetzt auf Wunsch asynchron auf dem Server — kein dauerhafter HTTP-Stream mehr nötig. Managed Agents können jetzt auch Remote-MCP-Server direkt einbinden, Custom Functions auf Client-Seite ausführen und ablaufende Credentials mittendrin austauschen, ohne den Sandbox-Zustand zu verlieren. Alles ist auf dem Free-Tier verfügbar und ab sofort über die Gemini Interactions API aktiv. Wer noch eigene Proxy-Middleware für den MCP-Zugriff baut, kann das jetzt weglassen.

English

Google has significantly expanded Managed Agents in the Gemini API: agent interactions can now run asynchronously on the server — no need to keep an HTTP connection alive for long-running tasks. Agents can also connect directly to remote MCP servers for private databases and internal APIs, run custom client-side functions alongside server-side tools, and rotate expiring credentials mid-session without losing sandbox filesystem state. Everything is available on the free tier via the Gemini Interactions API. If you're still writing proxy middleware to reach MCP from Gemini, you can drop it.

💡 Hack des Tages · Tip of the day

Claude Code 2.1.210 · Laufzeitzähler + Barrierefreiheitsmodus
Deutsch

Claude Code 2.1.210 bringt zwei nützliche Verbesserungen für den Alltag: Ein Live-Laufzeitzähler zeigt in der eingeklappten Tool-Summary-Zeile die verstrichene Zeit an — kein Rätselraten mehr, ob der Agent noch läuft oder hängt. Dazu kommt ein neuer Barrierefreiheitsmodus, der die Ausgabe auf reinen Plaintext umschaltet — ideal für Screen-Reader-Nutzer und Setups ohne ANSI-Farben. Wer Permission-Regeln konfiguriert, bekommt außerdem beim Start eine Warnung, wenn bestimmte Tool-Typen in den Allow-Regeln stehen, die besser durch ihre sichereren Gegenstücke ersetzt werden sollten.

English

Claude Code 2.1.210 ships two practical daily improvements: a live elapsed-time counter now ticks on the collapsed tool summary line — no more guessing whether the agent is still running or has frozen. There's also a new accessibility mode that switches rendering to plain text — perfect for screen readers and no-ANSI setups. And if you configure permission rules, a startup warning now flags when certain tool types appear in your allow rules that should be replaced with their safer counterparts.
