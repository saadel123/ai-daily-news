English

A brand-new init command scaffolds a fresh plugin right where it belongs so you're ready in seconds. The plugin slash command also gains autocomplete for all subcommands, installed plugin names, and known marketplace entries. And the agents dispatch now suggests native slash commands and bundled skills, no longer limited to project-only skills.

English

Claude Code 2.1.158 brings Auto mode to cloud deployments: Bedrock, Vertex and Foundry users can now activate it with a single environment variable. That means automatic model selection per task — Claude picks autonomously between Opus 4.7 and 4.8 depending on what the job needs.
