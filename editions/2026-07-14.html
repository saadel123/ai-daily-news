💡 Hack des Tages · Tip of the day

Claude Code 2.1.208 · Screen-Reader-Modus
Deutsch

Claude Code 2.1.208 bringt eine ganze Reihe von Accessibility-Verbesserungen. Die wichtigste: ein Screen-Reader-Modus, der die komplette Ausgabe auf Plain-Text reduziert, damit gängige Screen-Reader damit klarkommen. Dazu gibt es Vim-Insert-Remap-Support für Nutzer, die im Vim-Workflow bleiben wollen, und Mausklick-Unterstützung in Vollbild-Menüs. Den Modus aktivierst du per Flag beim Start, per Umgebungsvariable oder über einen Eintrag in der Settings-Datei — alle drei Wege funktionieren.

English

Claude Code 2.1.208 ships a set of accessibility improvements. The headline feature: screen reader mode, which strips all output down to plain text so standard screen readers can parse it cleanly. There is also vim insert-mode remap support for users staying in a vim workflow, and mouse-click support in fullscreen menus. You can toggle screen reader mode with a flag at startup, an environment variable, or an entry in your settings file — all three work.

BEFEHL · COMMAND

enable plain-text rendering mode so screen readers can parse output cleanly

# Alternativ per Umgebungsvariable · Alternative via environment variable
