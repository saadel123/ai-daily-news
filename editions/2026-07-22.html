💡 Hack des Tages · Tip of the day

Claude Code 2.1.217 · Emoji-Shortcodes · Subagent-Limits
Deutsch

Claude Code 2.1.217 bringt zwei nützliche Neuerungen auf einmal. Im Prompt-Input funktionieren jetzt Emoji-Shortcodes: Gibst du einen Doppelpunkt gefolgt von einem Schlüsselwort ein, erscheint ein Vorschlag-Menü mit passenden Emojis — wer das abstellen möchte, setzt emojiCompletionEnabled in der Config auf false. Wichtiger für Pipelines: Es gibt jetzt eine harte Obergrenze für gleichzeitige Subagenten, standardmäßig 20, und Subagenten dürfen per Default keine eigenen Subagenten mehr spawnen. Wer tiefere Verschachtelungen braucht, öffnet das Limit über eine Env-Var — praktisch, um unkontrollierte Kaskaden in großen Ultracode-Runs zu verhindern.

English

Claude Code 2.1.217 ships two handy updates at once. The prompt input now supports emoji shortcode autocomplete — type a colon followed by a keyword and a suggestion menu appears; disable it with emojiCompletionEnabled set to false in your config. More important for pipelines: there's now a hard cap on concurrent subagents (default 20), and subagents can no longer spawn their own subagents by default. You can raise the nesting depth limit via an environment variable — useful for preventing uncontrolled spawn cascades during large Ultracode runs.

BEFEHL · COMMAND
CLAUDE_CODE_MAX_CONCURRENT_SUBAGENTS=<n> claude [task]
CLAUDE_CODE_MAX_SUBAGENT_SPAWN_DEPTH=<depth> claude [task]
CLAUDE_CODE_MAX_CONCURRENT_SUBAGENTS
max subagents allowed to run simultaneously (default: 20)
CLAUDE_CODE_MAX_SUBAGENT_SPAWN_DEPTH
how many levels deep subagents may spawn their own subagents (default: 0 = disabled)
# Limit hochsetzen für große parallele Runs · Raise cap for large parallel runs
$ CLAUDE_CODE_MAX_CONCURRENT_SUBAGENTS=40 claude "run the full test suite in parallel"

# Eine Spawn-Ebene erlauben · Allow one level of nested subagent spawning
$ CLAUDE_CODE_MAX_SUBAGENT_SPAWN_DEPTH=1 claude "ultracode: migrate the entire repo"
