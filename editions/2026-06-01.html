GitHub Copilot · AI Credits Billing ab heute
Deutsch

GitHub stellt Copilot heute auf ein tokenbasiertes Abrechnungsmodell um: Weg von der Flatrate, hin zu "GitHub AI Credits" — 1 Credit entspricht 0,01 $. Jeder bezahlte Plan bringt ein monatliches Credit-Budget mit, das sich nach dem Abo richtet: Pro+ mit 39 Credits, Business mit 19 Credits pro User monatlich. Code-Completions und Next-Edit-Suggestions bleiben unlimited und verbrauchen keine Credits — der Billing-Schock trifft nur KI-intensive Aktionen wie Chat und Agents. Admins bekommen Budget-Controls auf Enterprise-, Team- und User-Ebene; wer das Budget ausschöpft, entscheidet selbst, ob weitere Nutzung auf eigene Kosten zugelassen wird.

English

GitHub is switching Copilot to usage-based billing starting today: goodbye flat subscription, hello AI Credits — 1 credit equals $0.01. Every paid plan ships with a monthly credit pool sized to the tier: Pro+ at $39, Business at $19 per user per month. Code completions and next-edit suggestions stay unlimited and don't consume credits — billing shock only hits AI-intensive actions like chat and agents. Admins get budget controls at enterprise, team, and user level; once the pool runs out, you decide whether usage keeps running at metered pay-as-you-go or stops.

Claude Code 2.1.158 · Auto Mode auf Bedrock, Vertex und Foundry
Deutsch

Claude Code bringt Auto-Mode jetzt auch auf unternehmenseigene Deployments — AWS Bedrock, Google Vertex und Azure Foundry. Aktiviert wird der Modus per Umgebungsvariable, die man entweder in der Shell-Session setzt oder dauerhaft in der Konfiguration verankert. Der Auto-Mode lässt Claude eigenständig entscheiden, welche Tools er aufruft, ohne bei jedem Schritt um Erlaubnis zu fragen — nützlich für vollautomatisierte Pipelines. Damit steht das Feature, das bisher exklusiv in der direkten Anthropic-Cloud lief, jetzt auch für Teams bereit, die ihren Compute selbst betreiben.

English

Claude Code now brings Auto mode to enterprise-owned deployments on AWS Bedrock, Google Vertex, and Azure Foundry. Activate it by setting an environment variable — either export it for the current session or anchor it permanently in your shell config. Auto mode lets Claude decide which tools to call without prompting for permission at every step, useful for fully automated pipelines. The feature, previously exclusive to direct Anthropic Cloud connections, is now available to any team running their own compute infrastructure.
