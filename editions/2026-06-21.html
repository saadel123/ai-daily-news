English

DeepMind has already built an internal prototype that asynchronously monitors coding agent trajectories and escalates suspicious actions for human review. Importantly: this is not a reaction to any real incident — it's entirely precautionary.

English

Claude Code 2.1.181 shipped an environment variable that most people missed: point it at a marker file and Claude Code knows you're at your machine — and stops sending you mobile push notifications for as long as the file exists. Handy for daily work: no phone buzzing when you're already at the terminal. Delete the file and push notifications on your phone automatically resume.

BEFEHL · COMMAND

# Einrichten: Marker-Datei anlegen und Variable setzen · Set up: create marker and export var

# Push wieder aktivieren: Datei löschen · Re-enable push: remove the marker file
$ rm ~/.claude-presence
