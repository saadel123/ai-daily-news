Claude Code 2.1.212 · /fork im Hintergrund
Deutsch

/fork erzeugt ab sofort eine Kopie der laufenden Session im Hintergrund, während du in der aktuellen weitermachst — der alte /fork, der einen In-Session-Subagenten öffnete, heißt ab sofort /subtask. Dazu kommt /resume als Session-Picker für vergangene und gelöschte Sessions. Neu: ein Session-weites Limit von 200 WebSearch-Aufrufen und 200 Subagenten-Spawns — damit kommen Endlosschleifen endlich an eine Wand. MCP-Tool-Calls, die länger als zwei Minuten dauern, wandern automatisch in den Hintergrund, damit die Session nutzbar bleibt.

English

/fork now creates a copy of the running session as a background session while you keep working in the current one — the old /fork behavior (launching an in-session subagent) moves to /subtask. /resume opens a picker of past sessions, including deleted ones, and resumes them as background sessions. New: a session-wide cap of 200 WebSearch calls and 200 subagent spawns, so runaway loops finally hit a wall. MCP tool calls that run longer than two minutes are automatically moved to the background so the session stays usable.

BEFEHL · COMMAND
/fork
/subtask [prompt]
/resume
claude auto-mode reset [--yes]
--yes
skip the confirmation prompt when resetting auto-mode to defaults

PrismML Bonsai 27B · iPhone-tauglich
Deutsch

Das Caltech-Startup PrismML hat Bonsai 27B veröffentlicht — ein multimodales 27-Mrd.-Parameter-Modell, das per 1-Bit-Quantisierung auf 3,9 GB schrumpft und auf dem iPhone 17 Pro mit rund 11 Tokens pro Sekunde läuft. Die Basis ist Qwen3.6 27B; Text und Bilder werden akzeptiert, das Modell ist unter Apache 2.0 kostenlos verfügbar. Apple soll die Technologie aktiv evaluieren. Trotz der extremen Kompression erreicht die 1-Bit-Variante noch über 90 Prozent der Full-Precision-Benchmark-Performance — ein ernstzunehmender Kandidat für lokale KI ganz ohne Cloud-Anbindung.

English

Caltech startup PrismML released Bonsai 27B — a multimodal 27B-parameter model that shrinks to 3.9 GB through 1-bit quantization and runs at roughly 11 tokens per second on an iPhone 17 Pro. It's based on Qwen3.6 27B, accepts text and images, and is freely available under Apache 2.0. Apple is reportedly evaluating the technology. Despite the extreme compression, the 1-bit variant still achieves over 90% of full-precision benchmark performance — a serious contender for local AI inference with no cloud dependency.
