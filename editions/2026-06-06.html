Claude Code 2.1.166 · fallbackModel
Deutsch

Claude Code 2.1.166 bringt den lang ersehnten Fallback-Schalter: In der settings.json oder per Flag lassen sich bis zu drei Ersatzmodelle in Reihenfolge definieren, die Claude automatisch versucht, wenn das Primärmodell überlastet oder nicht erreichbar ist. Wer viel mit Agentic-Pipelines arbeitet, kennt den Frust: ein kurzer Kapazitätsspike, und der gesamte Lauf hängt. Dank Fallback läuft die Arbeit still auf einem Alternativmodell weiter, ohne dass man eingreift. Ebenfalls neu: das thinking-Flag und die Env-Var deaktivieren jetzt auch bei Modellen, die standardmäßig denken, zuverlässig den Reasoning-Modus.

English

Claude Code 2.1.166 ships the long-awaited fallbackModel setting: define up to three backup models in order in settings.json or via the --fallback-model flag, and Claude will silently try them if the primary model is overloaded or unavailable. Anyone running agentic pipelines knows the frustration — a brief capacity spike and the whole run stalls. With fallback configured, work continues on the alternative model without manual intervention. Also new: the thinking flag and env var now reliably disable reasoning mode even on models that think by default.

BEFEHL · COMMAND
claude --fallback-model <model> [options]
--fallback-model <model>
model to try if the primary is overloaded or returns an unexpected error (repeatable, up to 3)
# Interaktive Session mit Fallback · Interactive session with one fallback
$ claude --fallback-model claude-sonnet-4-6

# Nicht-interaktiver Lauf mit Fallback · Non-interactive run with fallback
$ claude -p "Run migration check" --fallback-model claude-sonnet-4-6

ChatGPT Active Sessions · Session-Sicherheit
Deutsch

OpenAI hat Active Sessions in ChatGPT eingeführt — eine Sicherheitsfunktion, mit der Nutzer alle aktuell angemeldeten Sitzungen einsehen und gezielt oder komplett abmelden können. Unter Settings > Security > Active Sessions sieht man Gerät, App, ungefähren Standort, Anmeldezeit und ob es die aktive Sitzung ist. Damit lassen sich unbekannte oder vergessene Sessions sofort beenden — praktisch nach einem Gerätewechsel oder einem Credential-Leak. English

OpenAI has launched Active Sessions in ChatGPT — a security feature letting users review all currently logged-in sessions and sign out of individual ones or all at once. Under Settings > Security > Active Sessions you see device, app, approximate location, sign-in time, and whether it's the current session. That lets you instantly terminate unfamiliar or forgotten sessions — handy after a device change or a credential leak. One caveat: accounts using organizational SSO sign-in are excluded from the feature.

Claude Code 2.1.166 · --thinking disabled
Deutsch

Mit Claude Code 2.1.166 funktioniert ein bisher unzuverlässiges Toggle endlich sauber: Der Flag und die Env-Var schalten den Reasoning-Modus jetzt auch bei Modellen wie claude-opus-4-8 zuverlässig ab, die standardmäßig denken. In automatisierten Pipelines, wo man die Antwortlatenz minimieren oder schlicht Token-Kosten kontrollieren will, war das fehlende Toggle ein echter Schmerzpunkt. Jetzt kann man einen ganzen Claude-Code-Lauf mit einer einzigen Env-Var in den schnellen Instruct-Modus schalten.

English

Claude Code 2.1.166 fixes a previously unreliable toggle: the flag and env var now cleanly disable reasoning mode even on models that think by default, like claude-opus-4-8. For automated pipelines where you want to minimize response latency or control token spend, the broken toggle was a real pain. Now you can flip an entire Claude Code run into fast instruct mode with a single env var.

BEFEHL · COMMAND
claude --thinking disabled [options]
--thinking disabled
turn off extended reasoning for this session, even on models that think by default
MAX_THINKING_TOKENS=0
env-var equivalent — set in CI/CD or .env to disable thinking across all runs
# Reasoning für eine Session deaktivieren · Disable reasoning for one session
$ claude --thinking disabled

# Per Env-Var in CI-Pipelines · Via env var in CI pipelines
$ MAX_THINKING_TOKENS=0 claude -p "Run smoke tests and report"
