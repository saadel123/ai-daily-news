💡 Hack des Tages · Tip of the day

Claude Code 2.1.217 · Budget-Deckel, Subagenten-Limit & Emoji-Shortcuts
Deutsch

Claude Code 2.1.217 bringt drei Änderungen, die vor allem bei größeren Agent-Workflows ins Gewicht fallen. Erstens greift jetzt ein konfigurierbares Limit für parallele Subagenten — Standard sind 20, über eine Umgebungsvariable anpassbar. Zweitens spawnen Subagenten keine weiteren Subagenten mehr, solange du das nicht explizit erlaubst — das verhindert unkontrolliertes Wachstum in Orchestrierungs-Setups. Drittens stoppt der Budget-Deckel beim Start jetzt zuverlässig auch Background-Subagenten, die vorher durch die Maschen geschlüpft sind. Als Zugabe gibt es Emoji-Shortcode-Autocomplete im Prompt: tippe einfach einen Doppelpunkt gefolgt vom Emoji-Namen, und das Zeichen erscheint direkt.

English

Claude Code 2.1.217 brings three changes that matter most in larger agent workflows. First, there's now a configurable cap on concurrent subagents — default 20, tunable via environment variable. Second, subagents no longer spawn nested subagents unless you explicitly allow it, preventing runaway growth in orchestration setups. Third, the per-session budget flag now reliably stops background subagents too — something that slipped through before. Bonus: emoji shortcode autocomplete is live in the prompt input, so typing a colon followed by an emoji name inserts the character directly.
