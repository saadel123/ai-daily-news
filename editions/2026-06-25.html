💡 Hack des Tages · Tip of the day

Claude Code 2.1.191 · /rewind nach /clear
Deutsch

Claude Code 2.1.191 bringt einen lang gewünschten Trick: Wer mitten in einer Session den Chat leert, kann jetzt mit einem Slash-Command zurückspulen und genau dort weitermachen, wo er aufgehört hat — vor dem letzten Clear. Das ist besonders praktisch, wenn man die Kontext-History versehentlich weggefegt hat, aber den Gesprächsfaden nicht verlieren will. Außerdem merkt sich der aktuelle Build, welche Hosts in der Sandbox-Session mit Ja freigegeben wurden — kein ewiges Re-Approven mehr beim nächsten Tool-Call an denselben Host.

English

Claude Code 2.1.191 ships a long-wanted workflow trick: if you wipe the chat mid-session you can now use a slash command to rewind back to the state just before the last clear and pick up from there. Handy when you accidentally nuked your context history but still need that conversation thread. The build also remembers which sandbox hosts you approved with "Yes" for the entire session — no more re-approving the same host on every subsequent tool call.

BEFEHL · COMMAND
/rewind
# Sitzung vor dem letzten /clear wiederherstellen · Resume from before the last /clear
$ /rewind
