English

Claude Code picked up a string of quality-of-life improvements between versions 2.1.191 and 2.1.195. Biggest one: /rewind can now step back to conversation points before a /clear was run — if you accidentally nuked your context, you can re-enter instead of starting over. The bash-mode shortcut (the exclamation mark for running shell commands) now has live file path autocomplete, making terminal navigation noticeably smoother. On top of that, the team cut CPU usage during streaming responses by 37%, and the new autoMode.classifyAllShell setting routes all shell commands through the auto-mode classifier instead of only arbitrary-code-execution patterns.
