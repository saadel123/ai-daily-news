Claude Code 2.1.126 · project purge
Deutsch

Claude Code bringt mit Version 2.1.126 einen frischen Befehl: claude project purge [path] putzt den kompletten Claude-Code-Zustand für ein Projekt weg — Transcripts, Tasks, File-History, Config-Eintrag. Mit --dry-run kannst du erst mal anschauen, was rausfliegt, bevor du wirklich löschst. Praktisch, wenn du alte Repos entrümpeln willst oder ein Projekt von Grund auf neu setzen möchtest. Auch dabei: --all, -y/--yes und ein interaktiver Modus.

English

Claude Code 2.1.126 ships a brand-new command: claude project purge [path] deletes all Claude Code state for a project — transcripts, tasks, file history, the config entry. Pair it with --dry-run first to preview what would go before you actually wipe anything. Handy for cleaning up stale repos or starting a project's setup from a clean slate. It also supports --all, -y/--yes and an interactive mode.

# 1) Trockenlauf — zeigt, was geputzt würde, ohne wirklich zu löschen
$ claude project purge --dry-run ~/code/old-experiment

# 2) Wirklich putzen, ohne Rückfragen
$ claude project purge -y ~/code/old-experiment

💡 Hack des Tages · Tip of the day

Claude Code 2.1.128 · /color würfelt
Deutsch

Mit Claude Code 2.1.128 gibt's einen kleinen, aber fröhlichen Trick: /color ohne Argumente würfelt eine zufällige Session-Farbe. Praktisch, wenn du mehrere Sessions parallel offen hast und sie auf einen Blick auseinanderhalten willst. Bonus: /mcp zeigt jetzt die Tool-Anzahl pro Server und markiert Server, die mit 0 Tools verbunden sind — perfekt zum Debuggen kaputter Setups.

English

Claude Code 2.1.128 lands a small but fun tip: bare /color (no args) picks a random session color. Handy when you've got several sessions open in the same terminal layout and want to tell them apart at a glance. Bonus: /mcp now shows tool count per server and flags any server that connected with zero tools — perfect for spotting broken setups.

# 1) Würfelt eine zufällige Session-Farbe — easy to spot
$ /color

# 2) Zeigt Tool-Anzahl pro MCP-Server, flaggt 0-Tool-Server
$ /mcp
