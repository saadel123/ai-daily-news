English

Anthropic today adds two security-focused capabilities to Claude Managed Agents: MCP tunnels and self-hosted sandboxes. An MCP tunnel lets your agents reach private MCP servers on your internal network without opening a single inbound firewall port — a lightweight gateway makes one outbound connection, traffic is end-to-end encrypted, and everything stays behind your firewall. Self-hosted sandboxes keep the agent orchestration loop on Anthropic's side while you direct tool execution to your own infrastructure or to providers like Cloudflare, Daytona, Modal, or Vercel. For teams with compliance requirements, it's a clean architecture: sensitive data never leaves your perimeter.

Deutsch

Claude Code 2.1.144 räumt heute ein spürbares Workflow-Problem aus dem Weg: Der neue Slash-Command Resume holt jede laufende Background-Session direkt in den Vordergrund — sie erscheinen im Session-Picker neben den interaktiven Sessions, ein kleines bg-Badge markiert sie als Hintergrund-Job. Bonus: der frühere Extra-Usage-Command heißt jetzt Usage Credits — der alte Name funktioniert noch, aber der neue macht auf einen Blick klar, was der Befehl anzeigt.

English

Claude Code 2.1.144 resolves a real workflow friction point: a new slash command called Resume lets you jump into any running background session right from the prompt — background sessions appear in the session picker alongside interactive ones, flagged with a bg badge. The model command is now session-local: switching models only affects the current session. To persist a model choice as the default for new sessions, hit d in the model picker. Handy for teams running multiple agents with different models in parallel. Bonus: the old extra-usage command is now usage-credits — the old name still works, but the new one makes the purpose immediately obvious.
