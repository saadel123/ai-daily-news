English

Claude Code 2.1.199 brings a small but handy improvement: stacked slash-skill invocations now load all leading skills at once — up to five in a chain. Previously Claude Code only loaded the first skill in a chain and treated the rest as plain prompt text. If you need deep research and data visualization at the same time, you can invoke both skills together without typing them one at a time.

BEFEHL · COMMAND
/skill-1 /skill-2 [prompt]

$ /deep-research /dataviz Visualize AI lab funding rounds in H1 2026

# Bis zu 5 Skills in einer Kette · Chain up to 5 skills
$ /code-review /security-review /simplify Check this API handler
