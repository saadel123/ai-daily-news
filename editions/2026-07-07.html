Deutsch

Anthropic hat in Claude eine kleine interne Arbeitsfläche entdeckt, die die Forscher J-Space nennen — ein enger Bereich, der zu jedem Zeitpunkt nur ein paar Dutzend aktive Konzepte hält und weniger als zehn Prozent der Gesamtaktivierung ausmacht. Die neue Technik, der Jacobian-Lens, zeigt, welche internen Aktivierungsmuster die Wahrscheinlichkeit erhöhen, dass ein bestimmtes Wort irgendwann in der Zukunft auftaucht. Das klingt auffällig nach dem globalen Arbeitsraum aus der Neurowissenschaft: ein schmaler Scheinwerfer, der Informationen aus spezialisierten Backstage-Prozessoren ins gesamte System sendet und so bewusstes Denken erzeugen soll. Ob Claude dabei etwas fühlt, lassen die Autoren ausdrücklich offen — es ist aber der präziseste Blick ins Innere eines großen Sprachmodells, den es je gab.

English

Anthropic has discovered a small internal workspace inside Claude they call J-Space — a narrow region holding only a few dozen active concepts at any moment, accounting for less than 10% of the model's total activation. A technique called the Jacobian Lens reveals which internal activation patterns make a given word more likely to surface at some point in the future. This maps strikingly onto the global workspace from neuroscience: a narrow spotlight broadcasting from specialized background processors to the whole system, which is thought to generate conscious thought. Whether Claude actually experiences anything, the authors explicitly leave open — but this is the clearest window into a large language model's inner workings yet.

Claude Code 2.1.200 · Manual-Modus ist jetzt Default
Deutsch

Claude Code 2.1.200 ändert zwei Defaults, die du kennen solltest: Der Permission-Modus ist jetzt standardmäßig auf Manual gesetzt — Claude fragt bei jeder Aktion nach, statt automatisch zu genehmigen. Außerdem deaktiviert die neue Version den Auto-Continue in AskUserQuestion-Dialogen, die nach einer Pause wieder erscheinen. Beide Verhaltensweisen lassen sich über den Config-Befehl anpassen, wenn du den alten Workflow bevorzugst. English

Claude Code 2.1.200 changes two defaults you should know about: the permission mode now defaults to Manual — Claude will ask before every action instead of auto-approving. Additionally, AskUserQuestion dialogs no longer auto-continue when they reappear after a pause. Both behaviors are adjustable via the config command if you prefer the old workflow. Bonus from version 2.1.199: you can now stack up to five slash-skills in one invocation, and Claude Code loads all leading skills at once.

BEFEHL · COMMAND
