ChatGPT · Trusted Contact · Krisenfunktion
Deutsch

OpenAI hat für ChatGPT den Trusted Contact eingeführt: Wer möchte, kann in den Einstellungen eine erwachsene Vertrauensperson benennen, die von ChatGPT kontaktiert wird, wenn automatisierte Systeme und geschulte Reviewer ein ernstes Selbstverletzungsrisiko erkennen. Die Vertrauensperson erhält keine Gesprächsprotokolle — nur eine kurze Benachrichtigung per E-Mail, SMS oder In-App-Meldung. Das Feature ist freiwillig, global verfügbar (nicht für Minderjährige) und ersetzt professionelle Krisenberatung ausdrücklich nicht — aber es ist ein seltenes Zeichen, dass ein KI-Unternehmen an Sicherheit ohne Massenüberwachung denkt.

English

OpenAI introduced Trusted Contact for ChatGPT: users can optionally designate one trusted adult in their settings who may be notified if automated systems and trained reviewers detect a serious self-harm risk. The contact receives no conversation logs — just a brief notification via email, text, or in-app message. The feature is opt-in, globally available (not for minors), and OpenAI explicitly states it does not replace professional crisis services — but it's a rare sign of an AI company thinking about safety without mass surveillance.

💡 Hack des Tages · Tip of the day

Claude Code 2.1.129 · --plugin-url
Deutsch

Claude Code 2.1.129 bringt ein neues Flag, mit dem du eine Plugin-ZIP-Datei direkt aus einer URL für die aktuelle Session laden kannst — kein manueller Download, kein globales Installieren. Praktisch, wenn du ein Plugin schnell ausprobieren oder einem Kollegen eine URL schicken willst, ohne den normalen Publish-Flow zu durchlaufen. Bonus aus derselben Version: Ctrl+R durchsucht wieder standardmäßig alle Projekte und Sessions (wie vor Version 2.1.124), und über eine neue Umgebungsvariable lässt sich das automatische Upgrade via Homebrew oder WinGet einschalten.

English

Claude Code 2.1.129 ships a new flag that loads a plugin ZIP directly from a URL for the current session — no manual download, no global install required. Handy for quickly testing a plugin or sharing one with a colleague via a single URL, without going through the normal publish flow. Same-release bonus: Ctrl+R is back to searching all projects and sessions by default (as it was before 2.1.124), and a new environment variable lets Homebrew and WinGet users enable automatic background upgrades.

BEFEHL · COMMAND
