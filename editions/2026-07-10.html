💡 Hack des Tages · Tip of the day

Claude Code 2.1.206 · Agent-Monitor mit farbigen Status-Labels
Deutsch

Claude Code 2.1.206 überarbeitet den Agenten-Monitor komplett. Jede Zeile in der Übersicht zeigt jetzt ein farbiges Status-Wort — needs-input, working oder completed — plus eine vom Classifier generierte Zusammenfassung, was der Agent gerade tut, statt rohem Tool-Call-Text. Für blockierte Sessions öffnet das Peek-Fenster außerdem genau, was Claude von dir braucht. Mit Ctrl+X entfernst du abgeschlossene Sessions dauerhaft aus der Liste. Neu ist außerdem ein Gateway-Login-Endpoint: Mit dem Slash-Command Login kannst du dich jetzt direkt über Anthropic-betriebene Public-Gateway-Endpunkte authentifizieren, ohne über die Standard-API zu gehen.

English

Claude Code 2.1.206 fully overhauls the agent monitor. Every row now shows a colored status word — needs-input, working, or completed — plus a classifier-written summary of what the agent is doing, instead of raw tool call text. For blocked sessions, the peek view shows exactly what Claude needs from you. Ctrl+X permanently removes completed sessions from the list. Also new: a gateway login endpoint means you can now authenticate via Anthropic-operated public gateway endpoints directly from the Login slash command, without going through the standard API.
