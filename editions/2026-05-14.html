English

Claude Code 2.1.141 delivers three practical improvements: the new flag on the agent dashboard scopes the session list to a specific directory — no more scrolling through all projects, just the sessions for the current repo. The Rewind menu gains 'Summarize up to here', which compresses older turns without touching the fresh context turns. And background agents you launch with the keyboard shortcut now inherit your current permission mode instead of reverting to the default on start.

Deutsch

Wer täglich lange Claude-Code-Sessions führt, kennt das Problem: Der Kontext quillt über, Claude vergisst frühe Entscheidungen und Zusammenhänge. English

Anyone running long Claude Code sessions daily knows the problem: context overflows, Claude starts forgetting early decisions and context. The new 'Summarize up to here' option in the Rewind menu in version 2.1.141 bundles older turns into a compact summary — the fresh turns remain untouched. The context budget is freed up again and your current working state stays fully intact. Open the Rewind menu at the turn where you want to compress — done.
