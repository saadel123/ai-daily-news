💡 Hack des Tages · Tip of the day

Claude Code 2.1.181 · /config direkt aus dem Prompt
Deutsch

Claude Code 2.1.181 bringt einen praktischen Slash-Command: Mit /config key=value kannst du jetzt jede Einstellung direkt aus dem Prompt heraus setzen — Thinking an oder aus, Verbose-Modus, das Modell, alles. Das funktioniert im interaktiven Modus, mit -p und im Remote Control. Es gibt außerdem eine neue Umgebungsvariable CLAUDE_CLIENT_PRESENCE_FILE: wenn du eine Marker-Datei anlegst und ihren Pfad dort einträgst, unterdrückt Claude Code Mobile-Push-Benachrichtigungen, solange du am Rechner sitzt — kein ungewolltes Klingeln mehr auf dem Handy.

English

Claude Code 2.1.181 ships a handy slash command: /config key=value lets you set any setting directly from the prompt — thinking on or off, verbose mode, the model, anything. It works in interactive mode, with -p, and in Remote Control. There's also a new environment variable CLAUDE_CLIENT_PRESENCE_FILE: point it at a marker file and Claude Code will suppress mobile push notifications while you're at the machine — no more unwanted buzzing on your phone.

BEFEHL · COMMAND
/config key=value
# Thinking deaktivieren · Disable thinking
$ /config thinking=false

# Modell wechseln · Switch model
$ /config model=claude-opus-4-8

# Mobile-Pushs unterdrücken (Shell) · Suppress mobile push notifications (shell)
$ export CLAUDE_CLIENT_PRESENCE_FILE=~/.at-my-desk
