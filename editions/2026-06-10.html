💡 Hack des Tages · Tip of the day

Claude Code 2.1.169 · /cd · disableBundledSkills
Deutsch

Claude Code 2.1.169 hat im Schatten der --safe-mode-Schlagzeile zwei praktische Extras versteckt. Der Slash-Command /cd wechselt das Arbeitsverzeichnis einer laufenden Session, ohne den Prompt-Cache zu zerreißen — praktisch, wenn du mitten im Gespräch von einem Repo zum nächsten schwenkst und den Kontext behalten willst. Außerdem gibt es eine neue Einstellung, die alle eingebauten Slash-Commands, Workflows und Skills aus der Modellansicht ausblendet, wenn du eine schlankere Umgebung bevorzugst.

English

Claude Code 2.1.169 hid two handy extras in the shadow of its --safe-mode headline. The /cd slash command changes the working directory of a running session without tearing the prompt cache — handy when you pivot from one repo to another mid-conversation and want to keep your context intact. There's also a new setting that hides all built-in slash commands, workflows, and skills from the model's view if you prefer a leaner environment.

BEFEHL · COMMAND
/cd [path]
# Arbeitsverzeichnis wechseln ohne Cache-Verlust · Change directory without losing cache
$ /cd ~/projects/new-repo

# Zurück ins Home-Verzeichnis · Go back to home directory
$ /cd ~
