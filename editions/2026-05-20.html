💡 Hack des Tages · Tip of the day

Claude Code 2.1.145 · agents --json
Deutsch

Claude Code 2.1.145 bringt einen neuen Flag für den agents-Befehl: Die JSON-Ausgabe liefert alle laufenden Sessions als strukturiertes Objekt — ideal für Shell-Skripte, tmux-Resurrect-Integrationen oder eigene Status-Bars im Terminal. Das Tab-Fenster des Agent-Views zeigt jetzt außerdem die Anzahl der wartenden Eingaben an — so weißt du auf einen Blick, wann eine Session deine Aufmerksamkeit braucht, ohne das Fenster fokussiert halten zu müssen. Der Plugin-Browser listet jetzt alle Commands, Agents, Skills, Hooks und MCP-Server eines Plugins vor der Installation auf — kein blindes Einrichten mehr. Und der Status-Line-JSON-Feed enthält jetzt GitHub-Repo- und PR-Informationen, wenn ein Repo erkannt wird.

English

Claude Code 2.1.145 ships a new flag for the agents command: JSON output mode dumps all live sessions as structured objects — perfect for shell scripts, tmux-resurrect setups, or custom terminal status bars. The agent view tab title now shows the count of sessions awaiting your input, so you can glance at an alt-tabbed window and instantly know whether something needs attention. The plugin browser now lists a plugin's full manifest — commands, agents, skills, hooks, and MCP servers — before you commit to installing. The status line JSON feed also gains GitHub repo and PR information when a repository is detected.

BEFEHL · COMMAND
claude agents [--json]
--json
output all live sessions as structured JSON for scripting
# Alle Sessions als JSON ausgeben · List all sessions as JSON
$ claude agents --json

# Wartende Sessions filtern · Filter sessions awaiting input
$ claude agents --json | jq '.[] | select(.state=="awaiting_input") | .name'
