English

Claude Code 2.1.152 rounds off the review-fix workflow: a new flag on the code-review command applies simplification, reuse, and efficiency findings directly to your working tree — reviewing and fixing in one step. The old simplify command is now an alias for this. There's also a new slash command to reload skills without restarting the session — handy for plugin developers who tweak skills on the fly. Claude Code also now automatically falls back to your configured fallback model when the primary is unavailable, instead of failing every request.

/simplify
/reload-skills

--comment

low | medium | high | max
effort level controlling review depth; defaults to medium
