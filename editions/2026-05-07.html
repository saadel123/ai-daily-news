English

Claude Code 2.1.132 makes hooks smarter: the session ID is now available as an environment variable in Bash subprocess environments, so your hook script always knows which session it's running in — useful when coordinating multiple parallel agents or workflows. There's also a new env switch to disable the fullscreen alternate-screen renderer entirely; handy for setups like Emacs eat or other terminals that prefer native scrollback. Both variables are simple to set in the shell or config.
