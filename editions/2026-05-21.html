OpenAI · Erdős-Vermutung widerlegt
Deutsch

Ein allgemeines OpenAI-Reasoning-Modell hat die Planar-Unit-Distance-Vermutung von Paul Erdős aus dem Jahr 1946 widerlegt: Es fand eine unendliche Klasse von Punktanordnungen, die polynomial mehr Einheitsabstände erzeugen als das bis dahin optimale Quadratgitter. Das Ergebnis wurde von externen Mathematikern unabhängig verifiziert — und ist umso bemerkenswerter, weil das Modell nicht für Mathematik spezialisiert wurde. Der Beweis nutzt Werkzeuge aus der algebraischen Zahlentheorie, konkret das Golod-Shafarevich-Theorem aus den 1960er-Jahren.

English

An OpenAI general-purpose reasoning model has disproved Paul Erdős's 1946 planar unit distance conjecture: it found an infinite family of point arrangements producing polynomially more unit-distance pairs than the square grid previously thought optimal. External mathematicians independently verified the result — making it all the more remarkable that the model was not trained specifically for mathematics. The proof draws on tools from algebraic number theory, specifically the 1960s Golod-Shafarevich theorem.

💡 Hack des Tages · Tip of the day

Claude Code 2.1.145 · claude agents --json
Deutsch

Claude Code 2.1.145 macht Session-Monitoring richtig skriptfähig: Der neue JSON-Flag für claude agents liefert alle laufenden Sessions als maschinell auswertbares JSON — ideal für tmux-resurrect, eigene Dashboards oder Session-Picker. Dazu zeigt der Tab-Titel jetzt die Anzahl wartender Agenten, sodass du auf einen Blick siehst, wer auf dich wartet, ohne den Fokus zu wechseln. Und der Discover-Screen im Plugin-Browser zeigt Befehle, Agenten, Skills, Hooks und MCP-Server eines Plugins bereits vor der Installation — kein Blindkauf mehr.

English

Claude Code 2.1.145 makes session monitoring truly scriptable: the new JSON flag for claude agents dumps all live sessions as machine-readable JSON — perfect for tmux-resurrect, custom dashboards, or session pickers. The agents tab title now shows how many sessions are waiting for your input, so you know at a glance without switching context. And the Plugin Discovery screen now previews every command, agent, skill, hook, and MCP server a plugin provides before you install — no more buying blind.

BEFEHL · COMMAND
claude agents [options]
--json
output all live sessions as machine-readable JSON (agent_id, status, model, cwd)
# Alle Sessions als JSON ausgeben · Dump all live sessions as JSON
$ claude agents --json

# Nur wartende Sessions herausfiltern · Filter to sessions waiting for input
$ claude agents --json | jq '.[] | select(.status == "waiting")'
