💡 Hack des Tages · Tip of the day

Claude Code 2.1.149 · /usage nach Kategorie
Deutsch

Claude Code 2.1.149 macht das Limit-Tracking endlich aussagekräftig: Der Slash-Command zeigt jetzt, wie sich dein Verbrauch auf Skills, Sub-Agenten, Plugins und einzelne MCP-Server aufschlüsselt. Bisher war nur die Gesamtzahl sichtbar — jetzt kannst du aufspüren, welches Plugin deinen Kontext frisst oder welcher Sub-Agent besonders viel Compute verbraucht. Bonus: den Diff-View kannst du jetzt mit der Tastatur scrollen, und Markdown rendert GFM-Task-Checkboxen korrekt statt als Fließtext.

English

Claude Code 2.1.149 makes limit tracking genuinely useful: the /usage command now shows a per-category breakdown — skills, subagents, plugins, and individual MCP servers each get their own line. Previously you only got a total; now you can spot exactly which plugin is eating your context or which subagent is the most expensive. Bonus: the diff view can now be scrolled with keyboard shortcuts, and Markdown renders GFM task-list checkboxes properly instead of as plain text.
