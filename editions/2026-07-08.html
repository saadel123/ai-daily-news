Claude Code 2.1.203 · Login-Warning, Pause-Badge & MCP-Roots
Deutsch

English

Claude Code 2.1.203 brings three quiet but useful improvements. First, you'll now get a timely warning before your login expires — so background sessions don't freeze mid-task and force a re-auth scramble. Second, a grey ⏸ badge in the footer always shows whether manual permission mode is active — no more guessing. Third, all additional working directories in the session are now forwarded to connected MCP servers via roots/list, including change notifications so MCP servers automatically stay oriented in a monorepo.
