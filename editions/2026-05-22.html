Claude Code 2.1.147 · Pinned Sessions
Deutsch

Claude Code 2.1.147 bringt eine echte Neuerung für alle, die mehrere Hintergrund-Sessions parallel laufen lassen: Mit einem Tastendruck im Agenten-View heftest du eine Session an den laufenden Prozess. Wer gerade einen langen Agenten-Run läuft, verliert so keinen Kontext mehr. Unter Speicherdruck räumt Claude Code unangeheftete Sessions zuerst weg — angeheftete bleiben, bis du sie selbst schließt.

English

Claude Code 2.1.147 brings a genuine quality-of-life upgrade for anyone running multiple background sessions in parallel: a single keypress in the agent view pins a session to the live process. Pinned sessions stay alive even when Claude Code is idle, and during updates they are restarted in place rather than from scratch — so you lose no context. Running a long agent job? You now get a real pause-and-resume guarantee. Under memory pressure, Claude Code evicts non-pinned sessions first — pinned ones stay until you close them yourself.

💡 Hack des Tages · Tip of the day

Advisor-Strategie · Haiku fragt, Opus antwortet
Deutsch

Beim Code-with-Claude-London-Event vorgestelltes Muster für kosteneffiziente Agenten: Anstatt jeden Agenten-Schritt mit Opus durchzuführen, arbeitest du mit einem günstigeren Executor-Modell wie Haiku oder Sonnet, das einfache Aufgaben selbst erledigt. Nur wenn ein Schritt zu komplex wird, konsultiert der Executor kurz einen Opus-Advisor. Die meisten Agenten-Schritte sind repetitiv und brauchen keine Frontier-Intelligenz — der Advisor übernimmt nur die wirklich kniffligen Fälle. Anthropics internes Team berichtet, dass die Methode Kosten und Fehlerquote gleichzeitig senkt.

English

A cost-efficient agent pattern introduced at Code with Claude London: instead of routing every agent step through Opus, run a cheaper executor model — Haiku or Sonnet — that handles straightforward tasks itself. Only when a step gets too complex does the executor briefly consult an Opus-class advisor. Most agent steps are repetitive and don't need frontier-level intelligence — the advisor only steps in for the genuinely tricky cases. Anthropic's internal team reports that this approach reduces both cost and error rate at the same time.
