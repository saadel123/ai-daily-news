💡 Hack des Tages · Tip of the day

Claude Code 2.1.208 · Screen-Reader-Modus für Barrierefreiheit
Deutsch

Claude Code 2.1.208 bringt einen eingebauten Barrierefreiheitsmodus für Screenreader wie VoiceOver und NVDA: Spinner, ANSI-Farben und animierte Updates verschwinden, und stattdessen kommt klarer, linearer Plaintext raus. Ein Screenreader kann so sauber durch jede Ausgabe navigieren, ohne an Steuersequenzen zu hängen. Du kannst den Modus beim Starten per Flag aktivieren, als Umgebungsvariable in deiner Shell-Config hinterlegen oder dauerhaft in der Konfigurationsdatei einschalten.

English

Claude Code 2.1.208 ships a built-in accessibility mode for screen reader users — VoiceOver, NVDA, JAWS and friends. Spinners, ANSI color codes and animated text updates are switched off, replaced with plain linear text so any screen reader can navigate the output cleanly without tripping over control sequences. Set it at startup with a flag, export it as an environment variable in your shell config, or activate it permanently in settings.json.

BEFEHL · COMMAND
claude --ax-screen-reader
--ax-screen-reader
disable spinners, ANSI colors, and animations; output plain linear text
# Flag beim Start setzen · Set the flag at startup
$ claude --ax-screen-reader

# Dauerhaft als Umgebungsvariable · Permanently via env var
$ export CLAUDE_AX_SCREEN_READER=1
