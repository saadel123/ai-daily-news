English

OpenAI introduced three new audio models for the API: GPT-Realtime-2, GPT-Realtime-Translate, and GPT-Realtime-Whisper. The headliner is GPT-Realtime-2 — a single model that takes audio in and puts audio out, with GPT-5-class reasoning happening inside the audio loop rather than between transcription and synthesis steps. 💡 Hack des Tages · Tip of the day

Claude Code 2.1.129 · Plugins direkt aus dem Web
Deutsch

Claude Code 2.1.129 bringt einen neuen Flag, mit dem du ein Plugin direkt als ZIP-Archiv von einer URL laden kannst — ohne den Code erst lokal herunterladen zu müssen. Besonders nützlich in einer CI/CD-Pipeline oder auf einer Remote-Maschine, auf der du kein Dateisystem verwalten willst. Außerdem meldet Claude Code jetzt automatisch, wenn über Homebrew oder WinGet ein Update bereitsteht, und bietet an, die neue Version im Hintergrund zu installieren. Wer mehrere Setups betreibt, hat damit ein deutlich entspannteres Update-Erlebnis.

English

Claude Code 2.1.129 ships a new flag that lets you load a plugin directly as a ZIP archive from a URL — no need to download the code locally first. Especially useful in a CI/CD pipeline or on a remote machine where you don't want to manage a filesystem. Claude Code also now notifies you automatically when an update is available via Homebrew or WinGet, offering to install the new version in the background. Managing multiple setups just got a lot less tedious.

BEFEHL · COMMAND
claude --plugin-url <url> [options]
--plugin-url <url>
fetch a plugin .zip archive from a URL for the current session

$ claude --plugin-url https://example.com/my-plugin.zip

# Mit Modell-Auswahl kombinieren · Combine with model selection
$ claude --plugin-url https://example.com/my-plugin.zip -m claude-opus-4-7
