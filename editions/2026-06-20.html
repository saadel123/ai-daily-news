💡 Hack des Tages · Tip of the day

Hack · CLAUDE_CLIENT_PRESENCE_FILE · kein Handy-Ping mehr
Deutsch

Claude Code 2.1.181 hat eine Umgebungsvariable mitgebracht, die viele Nutzer übersehen haben: Trägst du in deiner Shell-Config den Pfad zu einer Marker-Datei ein, die existiert, solange du am Rechner bist, schaltet Claude Code die Mobile-Push-Benachrichtigungen automatisch stumm — kein Ping mehr auf dem Handy, während du ohnehin im Terminal sitzt. Das Einrichten dauert eine Minute: Datei anlegen, Variable dauerhaft in die Shell-Config eintragen, fertig. Wer mehrere Maschinen nutzt, kann die Variable je nach Rechner unterschiedlich konfigurieren — auf dem Laptop aktiv, auf dem Server-Account ohne.

English

Claude Code 2.1.181 shipped an environment variable that many users missed: set it in your shell config pointing to a marker file that exists while you're at your machine, and Claude Code automatically silences mobile push notifications — no more pings on your phone while you're already in the terminal. Setup takes a minute: create the file, add the variable permanently to your shell config, done. Users on multiple machines can configure it differently per machine — active on the laptop, absent on the server account.

BEFEHL · COMMAND
export CLAUDE_CLIENT_PRESENCE_FILE=/path/to/marker-file
CLAUDE_CLIENT_PRESENCE_FILE
path to a file that, when it exists on disk, tells Claude Code you are at your machine — mobile push notifications are suppressed while the file is present
# Marker-Datei anlegen und Variable dauerhaft setzen (zsh) · Create file and set var permanently (zsh)
$ touch ~/.claude-present && echo 'export CLAUDE_CLIENT_PRESENCE_FILE="$HOME/.claude-present"' >> ~/.zshrc

# Für eine einzelne Session deaktivieren · Disable for one session only
$ unset CLAUDE_CLIENT_PRESENCE_FILE
