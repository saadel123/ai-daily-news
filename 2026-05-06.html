🟧 Anthropic & Claude

Claude 4.7 Sonnet released
Deutsch

Anthropic hat heute Claude 4.7 Sonnet rausgebracht. Das neue Modell ist beim Coding rund 30 Prozent schneller als die Vorgängerversion und kostet trotzdem genauso viel. Besonders bei größeren Codebases läuft es deutlich zuverlässiger, weil Anthropic den Trainingsmix für Repository-Kontexte angepasst hat. Die ersten Reaktionen aus der Community sind durchweg positiv — viele Entwickler loben vor allem die bessere Fehlerdiagnose.

English

Anthropic shipped Claude 4.7 Sonnet today. The new model is roughly 30 percent faster on coding tasks than the previous version, while staying at the same price. It also handles larger codebases noticeably more reliably, since Anthropic adjusted the training mix for repository contexts. Early community feedback has been overwhelmingly positive — many developers especially highlight the improved error diagnosis.

→ anthropic.com/news/claude-4-7-sonnet
Memory API · Public Beta
Deutsch

Die neue Memory-API von Anthropic geht heute in die Public Beta. Sie speichert Kontext zwischen Sessions, sodass Claude sich an frühere Gespräche erinnert. Das ist vor allem für Agents praktisch, die über mehrere Tage hinweg an derselben Aufgabe arbeiten. Bisher musste man den Kontext bei jedem Aufruf manuell mitschicken, was schnell teuer wurde.

English

Anthropic's new Memory API moves into public beta today. It stores context across sessions, so Claude can recall earlier conversations. This is particularly handy for agents that work on the same task over several days. Previously you had to manually pass the context with every call, which got expensive fast.

→ docs.claude.com/memory

💡 Hack des Tages · Tip of the day

Prompt Caching
Deutsch

Mit Prompt Caching kannst du wiederholten Kontext um bis zu 90 Prozent billiger machen. Die Idee: Du markierst Teile deines Prompts als cacheable, und Anthropic speichert sie für bis zu fünf Minuten serverseitig. Beim nächsten Aufruf zahlst du nur noch einen Bruchteil. Wenn du immer denselben System-Prompt schickst, lohnt es sich auf jeden Fall.

English

With prompt caching, you can make repeated context up to 90 percent cheaper. The idea: you mark parts of your prompt as cacheable, and Anthropic stores them server-side for up to five minutes. On the next call, you pay just a fraction. If you always send the same system prompt, it's definitely worth it.

# Python — mark a long system prompt as cacheable
response = client.messages.create(
    model="claude-opus-4-7",
    system=[{
        "type": "text",
        "text": LONG_SYSTEM_PROMPT,
        "cache_control": {"type": "ephemeral"}
    }],
    messages=[{"role": "user", "content": question}]
)
→ docs.claude.com/prompt-caching
